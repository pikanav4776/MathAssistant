# MathAssistant v1.0.0 — Release Notes

**Product:** Algebra Co-Solving  
**Date:** June 2026

Use this file as the body for the GitHub release when tagging `v1.0.0`.

---

## Summary

First public-ready release of MathAssistant: a deterministic algebra step-validation tutor with multi-hop canonical paths, calculator UI, optional accounts, session resume, and production hardening.

---

## Highlights

- **Tutoring loop:** enter problem → submit steps → SymPy validation → error classification → escalating hints
- **Multi-hop paths:** distribute-then-combine, FOIL-then-combine (12 multihop problems regression-tested)
- **Calculator UI:** expression builder on problem entry and active session step entry
- **Auth:** register/login, guest mode, session ownership, signed-in session history
- **Session UX:** resume after refresh, session summaries on complete
- **Problem library:** 72 seeded problems, 12 curated starter problems with filters
- **Quality:** 100% classifier accuracy on 144-entry evaluation dataset; 549+ backend tests
- **Ops:** rate limiting, Render JWT guard, CI migration drift check, Alembic on deploy

---

## Breaking changes

None from v0.5 for API consumers; product version label unified to v1.0.
