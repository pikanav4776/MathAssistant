### Goal

Make sure your current FastAPI + SymPy system is **correct, reliable, and testable**

### You do:

1. **Fix and verify parsing**

   * 10–20 hand-written test cases

     * implicit multiplication (`2x → 2*x`)
     * parentheses multiplication
     * exponent parsing

2. **Test normalization + equivalence**

   * ensure:

     * `2(x+3) == 2x+6`

3. **Test classification logic**

   * confirm:

     * sign errors detected correctly
     * arithmetic errors detected correctly
     * distribution errors detected correctly

4. **Frontend sanity test**

   * input → submit → response renders correctly
   * verify correct vs incorrect UI paths
